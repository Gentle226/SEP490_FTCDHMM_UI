# Getting Started

Welcome to your new frontend starter project! This template provides a solid foundation for building modern React applications.

## What's Included

- **Next.js 15** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **shadcn/ui** component library
- **TanStack Query** for data fetching
- **next-intl** for internationalization
- **Storybook** for component development
- **ESLint, Prettier, Husky** for code quality

## Quick Start

1. **Install dependencies**:
   ```bash
   pnpm install
   ```

2. **Start development server**:
   ```bash
   pnpm dev
   ```

3. **Open your browser** and navigate to [http://localhost:3000](http://localhost:3000)

## Project Structure

The project follows a modular architecture:

- `src/app/` - Next.js App Router pages
- `src/base/` - Core infrastructure (components, hooks, utils)
- `src/modules/` - Feature modules (auth, users, etc.)

## Start Building

You can now start building your application by:

1. Creating new modules in `src/modules/`
2. Adding pages in `src/app/`
3. Using shared components from `src/base/components/`
